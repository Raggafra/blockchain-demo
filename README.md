#### Source

You'll need [Git](https://git-scm.com) to run the project from source. From your
command line:

```bash
# Clone this repository
$ git clone https://github.com/Raggafra/blockchain-demo

# Go into the repository
$ cd blockchain-demo

# Install dependencies
$ npm install

# Run the app
$ npm start
```
